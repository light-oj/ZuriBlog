
# ZuriBlog

This is sample blog project created as part of Zuri training.


## Deployment

To deploy this project run

```
  clone this repo
  
  cd <project_file>
  
  pipenv shell

  python manage.py runserver
```

  
## Running Tests

To run tests, run the following command

```
  python manage.py test
```
